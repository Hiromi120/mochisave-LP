# モチセーブLP 改良版

## 追加したこと

1. `content.js` の `design` で文字色・メインカラー・文字サイズを変更できます。
2. `content.js` の文章内で `\n` を使うと改行できます。
3. 文章内で次の強調指定が使えます。

```js
title: "[[追加料金なし]]で安心"
text: "{{福岡県全域対応}}です"
lead: "**見積り無料**\n最短即日対応"
subCopy: "《丁寧》・《安心》・明朗会計"
```

表示イメージ：

- `[[文字]]` → 黄色マーカー風
- `{{文字}}` → アクセントカラー
- `**文字**` → 太字
- `《文字》` → 文字の上に点々（圏点）
- `\n` → 改行

## デザインタイプ変更

`content.js` のここを変更します。

```js
design: {
  designType: "premium",
}
```

使える値：

- `premium`：高級感・上品
- `clean`：清潔感・大手感
- `friendly`：親しみ・地域密着

## 色変更

```js
colors: {
  main: "#032B6C",
  sub: "#2FA9F5",
  accent: "#FFD95A",
}
```

## 文字サイズ変更

```js
fonts: {
  heroLead: "22px",
  heroTitle: "56px",
  sectionTitle: "18px",
  body: "12px",
  cardTitle: "14px",
  button: "15px"
}
```

## 注意

- URL・画像パス・アイコン名には `[[ ]]` や `{{ }}` は使わないでください。
- 文章部分だけに使ってください。

## 圏点（文字の上の点々）の使い方

`content.js` の文章部分で、点々を付けたい文字を `《 》` で囲みます。

```js
subCopy: "《丁寧》・《安心》・明朗会計で選ばれています！"
```

注意：ボタン文字や長い文章全体に使うと読みにくくなるため、2〜6文字程度の短い言葉に使うのがおすすめです。
